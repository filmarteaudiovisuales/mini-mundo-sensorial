Mini Mundo Sensorial 🌈

Juegos simples, coloridos y sonoros para compartir con peques de 0 a 3 años.

Una experiencia para jugar con acompañamiento adulto, estimular la curiosidad y descubrir formas, sonidos y animalitos.

Guía para adultos 👪

Mini Mundo Sensorial está pensado para jugar en compañía de una persona adulta. La idea es compartir, nombrar colores, formas, sonidos y celebrar cada descubrimiento.

Algunas recomendaciones

  • 🕐 Usar sesiones cortas.
  • 🗣️ Acompañar con la voz.
  • 🤗 No usar como reemplazo del juego físico.
  • 🔊 Activar o desactivar sonidos según necesidad.
  • ◐ Usar el modo alto contraste si ayuda.
  • 🐢 Usar reducir movimiento si el peque se sobreestimula.

⚠️ Esta app no reemplaza orientación profesional, terapias, consultas médicas ni evaluaciones del desarrollo. Es solo una experiencia lúdica y de estimulación suave.
