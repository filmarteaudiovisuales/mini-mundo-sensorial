# mini-mundo-sensorial
Mini Mundo Sensorial es una app web pensada para el Día del Niño, pero usable durante todo el año. Está dirigida a familias, madres, padres, abuelos, jardines maternales, emprendedores infantiles y negocios que quieran ofrecer una experiencia digital simple, tierna e inclusiva para peques de 0 a 3 años.
